# Frog Dev Portfolio

A professional yet creative portfolio website built with a unique "dual-theme" approach, showcasing both professional technical background and creative web experiments.

## 🚀 Live Demo
Once hosted, you can view the site at: `https://[your-username].github.io/[repo-name]/`

## ✨ Features

- **Professional Homepage:** A clean, high-signal bio focusing on the journey from API & Integration Support to Full-Stack Development.
- **Artwork Gallery:** A responsive 2x2 grid (built with CSS Grid) featuring interactive hover states, zoom effects, and semi-transparent description overlays.
- **Web Design Playground (8-Bit):** A dedicated retro-themed space for apps and games, utilizing the 'Press Start 2P' font and pixel-perfect CSS styling.
- **Contact Interface:** A streamlined contact form and social link portal for professional inquiries.
- **Mobile First:** Fully responsive design across all pages, ensuring a seamless experience on any device.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Modern layouts using Flexbox and CSS Grid, plus custom 8-bit aesthetic styling.
- **JavaScript:** Client-side interactivity and logic.
- **Google Fonts:** 'Inter' for professional readability and 'Press Start 2P' for the retro playground.

## 📂 Project Structure

```text
├── index.html        # Professional Homepage

├── web-design.html   # 8-bit Web Design Playground
├── contact.html      # Contact Page
├── css/
│   ├── style.css     # Global Professional Styles
│   └── pixel.css     # Dedicated 8-Bit Theme Styles
├── js/
│   └── main.js       # Core Interactivity
├── assets/           # PDF Resume and static assets
└── images/           # Artwork and project images
```

## 🛠️ Local Development

1. Clone the repository:
   ```bash

   ```
